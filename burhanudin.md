## Project Repository

**Burhanudin.com - Personal Website**

- Build responsive personal portfolio using HTML5, CSS3, Tailwindcss and GSAP.
- Developed key pages including Home, Project and Contact.
- Implement modern UI Styling and Smooth animation using GSAP to enhance user experience.
- Github repository: [burhanudin.com](https://github.com/burhanudinrabbani666/burhanudin.com)

**Homie - Address Book App**

- Build a CRUD-based address book application using HTML5, CSS3, and JavaScript.
- Implemented core operations including create, read, update, and delete data.
- Applied clean code principles and basic data structure concepts for maintainable logic
- Github repository: [Homie](https://github.com/burhanudinrabbani666/homie)

**Better - Habit tracker and Daily Task App**

- Developed a habit tracker and simple daily task management app using TypeScript, React, Vite, Bun, and shadcn/ui.
- Implemented features for habit tracking, note-taking, and daily task organization.
- Designed reusable UI components and modern frontend architecture.

**Indonesian Wild Betta - REST API**

- Built a RESTful API providing data about Indonesian wild Betta species.
- Developed backend services using TypeScript, PostgreSQL, and Prisma ORM.
- Designed structured database schemas and implemented clean, maintainable API logic.
- Github repository: [Indonesia Wild Betta](https://github.com/burhanudinrabbani666/indonesia-wild-betta-api)
